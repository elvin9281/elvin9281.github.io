    Commands #

    紀錄常用的指令

    tar for .tar.gz #

    • 壓縮：
    tar zcvf FileName.tar.gz DirName
    • 解壓縮：
    tar zxvf FileName.tar.gz

    screen #

    • 開啟一個互動式的 Shell
    screen
    • Detach 互動式的 Shell
    Crtl + A + d
    • 列所有 互動式的 Shell(會列出 Session ID)
    screen -ls

    • Attach 到互動式的 Shell
    screen -r
screen -r + SessionID
    • 結束 互動式的 Shell
    exit

    uname - Get Kernel Version # 

    uname -a

    • 4 : Kernel version
    • 19 : Major revision
    • 0 : Minor revision
    • 17 : Patch level or number
